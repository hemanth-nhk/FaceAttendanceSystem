# Face Recognition Attendance Management System

A Python-based desktop application that automates attendance using real-time face recognition.

## Features
- Real-time face recognition using OpenCV and dlib
- PyQt5-based GUI
- Secure user registration with bcrypt
- Attendance stored using SQLite
- CSV and Excel report generation
- Offline, privacy-focused system

## Tech Stack
Python, OpenCV, face-recognition, PyQt5, SQLite, Pandas, PyInstaller

## Run Locally
pip install -r requirements.txt
python start.py

## Deployment
Packaged as a Windows executable using PyInstaller.
